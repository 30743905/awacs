## Header(Deprecated)

	Byte/     0        |       1       |       2       |       3       |
        /              |               |               |               |
       +---------------+---------------+---------------+---------------+
      0|             Magic             |    Version    |      Key      |
       +---------------+---------------+---------------+---------------+
      4|                           Body length                         |
       +---------------+---------------+---------------+---------------+
      8|                              PID                              |
       +---------------+---------------+---------------+---------------+
     12|                           Timestamp                           |
       +---------------+---------------+---------------+---------------+
       Total 16 bytes

- Magic: 0x5C4E

- Version: 0x01

- Key: user-defined
	
- Body length: body总长度(单位byte)

- PID: 进程号

- Timestamp: 秒级Unix时间戳
